# Golden Datasets

This directory contains **fully synthetic** evaluation data for the ADR AI Agent.

## Structure

```
data/golden/
  adr/
    generate_pdfs.py      # Generates synthetic ADR PDFs
    sample_adr_1.pdf      # Emergency visit — asthma exacerbation
    sample_adr_2.pdf      # Discharge summary — CHF
    sample_adr_3.pdf      # Post-surgical follow-up — knee replacement
    qa_pairs.json         # Expected Q&A for evaluation
  policy/
    generate_pdfs.py      # Generates synthetic policy PDFs
    sample_policy_1.pdf   # Asthma management policy
    sample_policy_2.pdf   # Cardiac rehabilitation policy
    sample_policy_3.pdf   # Total joint replacement policy
    qa_pairs.json         # Expected Q&A for evaluation
```

## No Real PHI

All documents are **synthetic** — generated by reportlab with invented patient data and policy text. Real clinical documents for production testing are stored in GCS and never committed to this repository.

## Regenerating PDFs

```bash
pip install reportlab
python data/golden/adr/generate_pdfs.py
python data/golden/policy/generate_pdfs.py
```

## Running Evaluation

```bash
python tests/eval/run_eval.py --api-url http://localhost:8000
```
